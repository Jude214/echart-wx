### echart-wx
echarts library compatible with wechat mini program.

### Install
```shell
$ npm install --save echart-wx
```

### Usage
## As an ES6 Module
```
import echarts from 'echart-wx'
```
## node.js (commonjs)
```
const echarts = require('echart-wx')
```

### Compiles and minifies for production
```
npm run build
```
